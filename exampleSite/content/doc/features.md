# Math 

The theme includes [KaTex 0.16.8](https://github.com/KaTeX/KaTeX), e.g. 

```tex
$$
\int_{-\infty}^{-\infty} \frac{1}{\sigma\sqrt{2\pi}}
    \exp\left(-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2{}\right)
$$
```

yields:


$$
\int_{-\infty}^{-\infty} \frac{1}{\sigma\sqrt{2\pi}}
    \exp\left(-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2{}\right)
$$
